¶ 与 PyCharm 集成

接下来，我们将开发环境与 PyCharm 整合，这样一来，后期的代码开发工作主要就在 PyCharm 上进行。

  1. 打开 PyCharm 程序，点击 File --> Open，将 PyCharmProjects/manabe 作为项目地址，当设置完成后，PyCharm 的编辑如下图

  2. 点击 PyCharm 首选设置，选择 Project: manabe --> Python Interpreter，然后在右边窗口，点击齿轮图表，选择 Show All…，在弹出的窗口中选择前面新建的虚拟环境 manabe 作为此项目的 Python interpreter，配置完后如下图所示

  3. 点击 Apply 以及 Ok。完成配置

和 Jenkins 类似，GitLab 也提供了操作 API 的官方文档：https://docs.gitlab.com/ee/api/。但这个操作是统一标准的，通过 REST API 的方式，支持主流的各种语言（Go，Java，JS，Python 等）。但为了在 Python 环境下更有效率，更规范，更自然地操作这些 API，我们这里也使用了一个第三方库 Python-GitLab 去操作这些 API。

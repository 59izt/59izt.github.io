快速安装指南

主要分以下步骤：

  • 安装 Python 3.x.x（Python 3.8.6）
  • 安装 Django（Django 3.1）
  • 设置数据库（MySQL 8.0.21）

¶ 安装 Python 和 Django

安装 Python 使用 pyenv install -v 3.8.6 安装即可，安装好 Python 后，使用 pip install django 即可自动安装最新版的 Django；
要想设置 Django 使用 MySQL 引擎，需要安装 mysqlclient，此处也是使用 pip install mysqlclient 安装；

安装 mysqlclient 过程可能遇到的问题：

  • 找不到 my_config 文件：
  • 找不到 _ctypes 模块

¶ 设置数据库使用 MySQL

  • 配置项目使用 MySQL 数据库，项目目录下的 settings.py 文件

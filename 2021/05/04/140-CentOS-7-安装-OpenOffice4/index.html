CentOS 7 安装配置 Apache OpenOffice 4.x
发表于 2021-05-04 | 更新于 2021-05-04
OpenOffice
Others | OpenOffice

¶ 安装 Apache OpenOffice

  1. 前往 Apache OpenOffice 官网下载对应的操作系统安装包；

  2. 解压安装包，得到 zh-CN 文件夹

  3. 进入 zh-CN/RPMS 文件夹，安装所有的 rpm 包

    安装完成后，OpenOffice 的安装路径在 /opt/openoffice4;

  4. 启动 OpenOffice

  5. 查看启动结果

¶ 拍错

  1. 错误一: no suitable windowing system found, exiting，解决方法如下

  2. 错误二: /opt/openoffice4/program/soffice.bin: error while loading shared libraries: libXext.so.6: cannot open shared object file: No such file or directory，解决方法如下

  3. 错误三: /opt/openoffice4/program/soffice.bin: error while loading shared libraries: libfreetype.so.6: cannot open shared object file: No such file or directory，解决方法如下

¶ 处理 Word 转 PDF 乱码问题

  1. 将 Windows 中常用的中文字体上传到 Linux 系统 /opt/openoffice4/share/fonts/truetype/ 目录

  2. 修改系统字体配置文件，添加 OpenOffice 字体库所在目录

  3. 安装字体管理工具

  4. 刷新字体缓存

  5. 查看安装的字体

  6. 重启 OpenOffice

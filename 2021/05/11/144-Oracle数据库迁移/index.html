¶ Oracle 迁移步骤

由于之前 Oracle 数据库时运行在容器中，现在需要将它迁移到宿主机上运行，迁移步骤如下：

  1. 先在宿主机上安装和容器中运行的 Oracle 的一致版本；需要注意字符集编码，实例名，服务名等；
  2. 停掉容器，停掉宿主机运行的数据库，将容器中映射出来的 oradata，flash_recovery_area 目录拷贝到宿主机的安装目录。比如 /u01/app/oracle；
  3. 启动数据库到 nomount 状态，并更改控制文件位置，然后关闭数据库；
  4. 启动数据库到 mount 模式；更改数据文件，日志文件，临时文件的位置；
  5. 打开数据库；
  6. 重启验证；

¶ 开始迁移

  1. 查看相关文件位置

  2. 关闭监听，关闭数据库

  3. 拷贝数据文件，控制文件，临时文件，日志文件等；

  4. 启动数据库到 nomount 模式，更改控制文件位置，然后关闭数据库

  5. 启动数据库到 mount 模式，更改数据文件，日志文件，临时文件的位置；

  6. 打开数据库

  7. 重启验证

¶ 实战 Django

我们现在将使用 Docker Compose 配置并运行一个 Django/PostgreSQL 应用。在一切工作开始之前，需要准备三个必要的文件：

  • Dockerfile: 指定Docker 容器要安装运行的内容;
  • requirements.txt: 指定容器内 Python 需要安装的模块;
  • docker-compose.yml: 将所有的内容关联起来，描述应用的构成。

¶ 编写 docker-compose.yml 文件

docker-compose.yml 文件将把所有的东西关联起来，它描述了应用的构成（一个 web 服务和一个数据库），使用 Docker 镜像，镜像之间的连接，挂载到容器的卷，以及服务开放的端口。

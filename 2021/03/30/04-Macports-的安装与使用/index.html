¶ MacPorts 常用命令

  • 更新 ports tree 和 MacPorts 版本： sudo port -v selfupdate
  • 搜索索引中的软件： port search name
  • 安装软件： port install name
  • 卸载软件： sudo port uninstall name
  • 查看有更新的软件以及版本： port outdated
  • 升级可以更新的软件： sudo port upgrade outdated

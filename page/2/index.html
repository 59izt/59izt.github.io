¶ Harbor 与 Registry

Harbor 是一个英文单词，意思是港湾，港湾是干什么的呢？就是停放货物的，而货物呢，是装在集装箱中的，说到集装箱，就不得不提到 Docker 容器，因为 Docker 容器的技术正是借鉴了集装箱的原理。所以，Harbor 正是一个用于存储 Docker 镜像的企业级 Registry。

menu.Mac 配置 Docker 镜像加速

参考文章：Docker 配置国内源加速镜像下载

由于网络原因，我们在pull Image 的时候，从Docker Hub上下载会很慢，所以可以使用国内的镜像（mirror）来加速,方便大家使用。

¶ 配置阿里云开发者平台

¶ 登录阿里开发者平台

  • 阿里开发者平台网址： https://promotion.aliyun.com/ntms/act/kubernetes.html#industry

¶ 点击镜像搜索

自动跳转到控制台的镜像搜索

¶ 注册/登录

注册或登录后，进入 Docker 镜像仓库（https://cr.console.aliyun.com/cn-hangzhou/instances/repositories），选中左边菜单栏的镜像加速器，如下图：

这里可以看到，系统已经为我们生成了一个专属加速器地址：https://xxxxx.mirror.aliyuncs.com

¶ 配置 Docker 服务

编辑 Docker 的守护配置文件 damon.json 文件

¶ 重启 Docker 服务

点击菜单栏上的 Docker 图标，选择 Restart，等待服务重启完成，即可使用镜像加速下载镜像。
